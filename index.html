Cara Menggunakan Viewer 3D

Model 3D Anda telah berhasil dimuat. Berikut adalah cara untuk berinteraksi dengan model:

  • Putar Model: Klik dan seret dengan mouse untuk memutar model
  • Zoom: Gunakan scroll mouse atau tombol zoom di atas untuk memperbesar/memperkecil
  • Geser: Tekan tombol Shift + klik dan seret untuk menggeser model
  • Putar Otomatis: Gunakan tombol "Putar Otomatis" untuk memutar model secara terus-menerus
  • Reset View: Klik "Reset View" untuk mengembalikan model ke posisi awal
